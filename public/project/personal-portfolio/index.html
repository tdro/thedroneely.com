Posted 1 June 2018 in projects; tagged with portfolio

Live Website

This is the website you’re browsing right now. The general structure is based on design patterns from the Laravel PHP framework. The comments are handled by Isso which is a backend commenting server written in Python. The static parts of this website are compiled by Hugo, while the PHP front controller handles the rest.

The backend traffic dashboard is integrated into the website using GoAccess which is a traffic monitor that reads the access logs of Nginx directly. Nothing much else to it. This website is meant to be small, lean, and compact since it runs on a tiny Raspberry Pi.
