Voiceover Website

Posted 7 May 2018 in projects; tagged with voiceover

Live Website

Voice acting is one of my interests, and I’ve made a website to manage just that. There were a few super simple HTML, CSS, & JavaScript websites that I created before this one but I consider this my first formal website that actually does something.

It’s made using a combination of PHP for dynamic content and Hugo for static content – a MySQL database for the backend, and Nginx as the web server. Everything sits behind a front controller and the project structure is based on a very barebones implementation of the Laravel framework.

To manipulate the content on the site, markdown files are edited and the static components of the website are recompiled by Hugo. The comment section is handled by a PHP commenting system called HashOver.
