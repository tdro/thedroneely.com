Bulma Resume

Posted 16 October 2018; tagged with bulma, css, framework, webdev


Live Website

When you search for “CSS Framework” in DuckDuckGo, Bing, or Google, one of the top results happens to be Bulma. This framework appears to be everyone’s new toy. I’ve been dabbling deeply with the Bulma CSS Framework on a current e-commerce project and it has been a pleasure to work with so far. This comes from someone who ends up breaking frameworks every so often.

Invariably, this is less of a project and more of a quick demonstration of modern resume design using Bulma. Something like this could take a bit of time to get right from scratch, but Bulma’s Flexbox based design makes it almost effortless.

It’s a simple web-page but the possibilities are endless. We can add some overlays, sprinkle in a bit of JavaScript or divvy up additional information into active tabs. In addition to Bulma, I’ve heard good things about Tachyons and Tailwind CSS, which are also excellent lightweight alternatives to Bootstrap.
